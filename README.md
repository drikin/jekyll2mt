jekyll2mt
=========

Post MT entry from Jekyll Markdown

Install
-------

``` sh
$ npm install
```

Setup
-----
Modify `MT_DATA_API_URL`, `MT_USER`, `MT_PASSWORD` in index.js for your environment;

Usage
-----

``` sh
$ node index.js -i <Jekyll markdown entry file path>
```
